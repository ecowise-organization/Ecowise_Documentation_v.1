# Events and volunteering

> Hosts post events; individuals register; the roster runs the day.

## Overview

Events and volunteering is Ecowise's social surface. Organisations, governments, and
administrators post events — workshops, beach cleanups, training days, fundraisers — that
appear in a community feed. Individuals discover events, register as attendees or
volunteers, leave comments, and react. Hosts run the registration roster — approve, decline,
mark check-in, no-show — and publish or cancel events as plans change.

It is the primary path for individuals to translate intent into in-person action.

## What it does

- Browse the event feed; filter by date, location, sector.
- Register as attendee or volunteer; withdraw.
- Comment, react, share.
- Host: create, edit, publish, cancel an event; manage roster decisions; upload cover media.

## Who uses it

Hosts (organisation, government, admin) and individuals on the attendee/volunteer side.

## Key features

- Cursor-paginated feed — the same shape on web and mobile.
- Roster decisions persist with timestamps for accountability.
- Event posts feed both the events tab and the dashboard discover section.
