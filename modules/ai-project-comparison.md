# AI project comparison

> A side-by-side, scored comparison of any two projects.

## Overview

Project comparison lets any user pit two projects against each other and get a structured,
scored side-by-side. The comparison covers mission alignment, scope, sustainability impact,
capability fit, and execution readiness. Each comparison is persisted so users can revisit
their history or share the result.

This is most useful for investors weighing where to pledge and for organisations
benchmarking against similar work.

## What it does

- Searches published projects to populate the comparison picker.
- Runs a head-to-head LLM comparison and persists the structured result.
- Shows comparison history per user (and per project for owners).

## Who uses it

Any authenticated organisation, individual, or government user.

## Key features

- No ownership requirement — an individual can compare two projects neither belongs to for
  due diligence.
- Persisted history means past comparisons remain a reference.
