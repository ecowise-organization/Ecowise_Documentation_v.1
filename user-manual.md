## Sign up

Pick the account type from the signup hub — Individual, Organisation, or Government — fill in
the required fields, then verify your email through the one-time code we send. Verified
accounts unlock the full feature set.

## Complete your profile

Open your profile page, add a photo, bio, location, and (for individuals) skills. A complete
profile improves your match scores.

## Create a project

Organisation accounts can create a project two ways:

- **Manually** — open the project edit form and fill in title, description, sector, and stage.
- **AI generator** — submit a short prompt and the platform produces the project, plan, and
  required skills in one round-trip.

## Run an AI analysis

From the project overview page, open the analysis card. The platform runs the analysis and
returns a tier, a list of insights, and an SDG mapping. Re-running keeps history.

## Receive matches

The dashboard's "for you" section pulls from the matching engine. Volunteers see projects;
project owners see candidate volunteers; issue owners see candidate solvers; task owners see
suggested assignees.

## Become an investor and pledge

Enable the Investor role from your account settings. The investor dashboard appears. Open a
project, click Pledge, set the amount and message. The project owner sees your pledge in their
inbox.

## Publish a government issue

Government accounts use the issue composer. Describe the problem, set sector and target
outcomes, then publish. Organisations can now propose solutions.

## Accept and promote a solution

When a government accepts a solution, the winning organisation gets an option to "promote" it
into a full project, carrying the solution copy forward and unlocking planning and matching.

## Execute a project plan

The plan page renders sections (serial or parallel) and tasks. Assign tasks, set due dates,
and let assignees move task status. The web client uses an interactive flow chart; the mobile
client uses a hierarchical list.

## Engage — messaging, events, RSVP

Open the messages tab to start a direct conversation or a group. Browse the events feed and
register for events that interest you. Your registrations and the events you host appear
automatically on the unified calendar.

## Generate a sustainability report

Government accounts open the report page and click Generate. The platform aggregates the
account's footprint into an ISO 26000 / GRI-aligned digest.
